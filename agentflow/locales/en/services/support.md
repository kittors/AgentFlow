# 辅助服务 (SupportService)

> 提供通用auxiliary functions。

## 功能

```yaml
file operations辅助:
  - 安全delete（备份afterdelete）
  - 批量重命名
  - 目录结构generate

Git 操作辅助:
  - 分支manage建议
  - commit message generate（BILINGUAL_COMMIT=1 when双语）
  - PR 描述generate

输出辅助:
  - Markdown 表格格式化
  - 代码块语法高亮
  - 树形结构渲染

错误restore:
  - 操作日志记录
  - 回滚建议
  - 断点续传statussave
```
